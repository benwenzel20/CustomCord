Custom Discord Theme

To use themes created you must have a the lastest version of BetterDiscord installed. To find that please go to BD Downloads and select which one you want. Then follow the link to the github and download the latest version.

If you don't want to create your own theme, you can find many premade themes here. Download the file and open the folder it downloaded to. Then open discord and go to settings. Scroll down to the bottom and click Themes (must have BD installed). Open the theme folder and drag the theme file you downloaded into your theme folder.
